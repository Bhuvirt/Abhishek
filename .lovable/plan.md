

# Abhishek Choudhary — Premium Portfolio Website

## Overview
A futuristic, immersive portfolio website for a Client Engagement & Operations Specialist. Dark theme with neon blue/violet accents, glassmorphism, smooth animations, and a 3D hero section.

## Design System
- **Font**: Inter
- **Colors**: Dark blue/black base (#0a0a1a), neon blue (#00d4ff), violet (#7c3aed) accents
- **UI Style**: Glassmorphism + soft glow effects
- **Layout**: Clean, spacious, premium feel

## Pages & Sections

### 1. Cinematic Preloader
- Fullscreen black screen with "Abhishek Choudhary" centered
- Animated progress bar (0→100%) using GSAP
- Preloader scales down and fades, revealing main site

### 2. Hero Section
- Embedded Spline 3D orb as fullscreen background
- "Hi, I'm Abhishek" headline with fade-in + blur animation
- Subtitle: "Client Engagement & Operations Specialist"
- Two CTA buttons: "View Experience" and "Contact Me" with neon hover glow
- The uploaded profile photo will be used in the About section

### 3. About Section
- Left: Profile image (uploaded photo) in glowing circular frame with hover lift/rotate
- Right: Bio text + key strengths displayed as icon cards with stagger animations
- Skills: Stakeholder Management, Client Communication, Process Coordination, Data Analysis, Recruitment Operations

### 4. Experience Timeline
- Vertical glowing timeline with scroll-triggered reveals
- Three roles: POD.ai (current), ServerGuy/Breeze.io, Zinple Info Solutions
- Each node has bullet points detailing achievements
- Alternating left/right layout on desktop, single-column on mobile

### 5. Skills Section
- Glowing card grid organized into Core Skills, Tools, and Soft Skills
- Hover effects with neon glow on each card

### 6. Work Approach Section
- Visual flow: Understand → Align → Execute → Deliver
- Connected glowing nodes with supporting tagline

### 7. Contact Section
- Glassmorphic form (Name, Email, Message) with glowing focus effects
- Email, Phone, LinkedIn links
- Submit button with pulse hover animation

### 8. Footer
- Minimal nav links, social icons, subtle glowing particles background

## Animation Strategy
- GSAP ScrollTrigger for all scroll-based animations
- Locomotive Scroll for smooth scrolling & parallax
- Stagger animations on skill cards, timeline items
- Floating glow background elements throughout

## Responsiveness
- All sections stack vertically on mobile
- Timeline becomes single-column
- CTAs remain prominent and accessible
- Spline 3D scales appropriately
